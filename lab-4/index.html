Addendum

Stylistic Choices

I chose a clean, modern sans-serif font (Roboto) for readability and professionalism. Headings use larger sizes and bold weight to clearly separate sections. The body text is 16px for comfortable reading. The color palette uses a light background (#f9f9f9) with dark text (#222) for contrast, and accent colors (navy blue for headers, teal for links) to guide attention. I added generous margins and padding around sections and between paragraphs to reduce visual clutter and improve flow.

HTML Abuse

In Lab 2, I originally considered using an <h2> tag for the essay title because it looked visually better, but I corrected it to <h1> since it is the highest-level heading. All visual adjustments are now handled in CSS, not by misusing HTML tags.
