Custom Font Demonstration

This page demonstrates how to serve a custom font directly from a web server using the @font-face CSS rule. The font used here is Rye, which is hosted locally in the fonts/ folder.

When a browser loads this page, it requests the font file from the server. Once the font is downloaded, it is applied to all the text on the page. This ensures a consistent look regardless of whether the user has the font installed on their system.

Note: The font file must be correctly referenced and accessible on the server for the text to render as intended.
