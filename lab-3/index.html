Serving a Custom Font

This page demonstrates how to serve a custom font to a browser. The font used here is Comic Neue, which is not commonly installed on most computers. We used the @font-face CSS rule to instruct the browser to download and apply this font.

When the browser loads this page, it makes a request to the server for the font file. Once downloaded, the browser renders all text using this font, giving the page a unique look.
