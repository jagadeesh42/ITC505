Interactive Number Sort

Enter numbers separated by commas, and click "Sort" to see them ordered.

Addendum

I created a **Holy Grail Layout** with CSS Grid. In the main section, I implemented an **interactive bubble sort tool** where users can input numbers and see them sorted. I solved this by writing a simple bubble sort function in JavaScript and connecting it to a button event. One difficulty was handling invalid input; I solved it by filtering out non-numeric values.
